Node.js

定义


Node.js是一个Javascript的运行环境，用C++语言编写而成

作用


编写系统级或者服务器端的Javascript代码，交给Node.js来解释执行

实现


采用了Google Chrome浏览器的V8引擎，性能很好，同时还提供了很多系统级的API，如文件操作、网络编程等

特性


事件驱动
异步编程
单进程单线程

安装


登陆官网http://nodejs.org/

command窗口查看node -v

npm


npm是Node.js默认的包管理器，集成到了Node.js的安装包里面
command窗口查看npm -v

用node.js实现简单的server


安装express和socket.Io库
npm install --save express
npm install --save socket.io
新建一个文件index.js
输入node index.js
浏览器输入http://localhost:8080/查看结果
